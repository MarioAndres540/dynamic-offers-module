# Frontend - Dynamic Offers Module

Interfaz de usuario moderna construida con React y Vite para la gestión de ofertas dinámicas.

## Tecnologías
- **React 18**
- **TypeScript**
- **Vite**
- **Tailwind CSS**
- **Axios**

## Scripts Disponibles
- `npm run dev`: Inicia el servidor de desarrollo de Vite.
- `npm run build`: Genera el bundle de producción en la carpeta `/dist`.
- `npm run preview`: Previsualiza la build de producción de forma local.

## Características
- Formulario dinámico para creación de separatas.
- Feedback en tiempo real sobre validaciones de negocio.
- Diseño responsivo y moderno.
